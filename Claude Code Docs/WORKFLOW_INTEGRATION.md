# Workflow Integration System

## Главная идея

Workflow является источником истины.

Не UI.

Не код.

Не конкретная модель.

---

# Схема

Workflow JSON

↓

Workflow Parser

↓

Adapter

↓

Generated UI

↓

ComfyUI Engine

---

# Подключение новой модели

Шаг 1

Получить workflow.

---

Шаг 2

Создать папку:

plugins/model_name

---

Шаг 3

Создать:

adapter.ts

schema.json

manifest.json

---

Шаг 4

Зарегистрировать плагин.

---

# Запрещено

Не создавать отдельный экран под каждую модель.

Не копировать существующий код интерфейса.

Не дублировать функциональность.

---

# Цель

Любая новая модель должна подключаться через систему адаптеров.

Время интеграции новой модели:

менее 1 часа.
