История изменений

15.11.2026 release

🚀 Релиз v2: Terminal Override

Скрипты полностью переписаны. Идемпотентность достигла 99%. При рестарте Kaggle окружение поднимается за ≈ 12 секунд вместо полной переустановки.

  • Новый kaggle_env.py: единый модуль путей, uv, ремонта venv.
  • instal_comfyui.py теперь идемпотентен по всем зависимостям.
  • start.py: проверки окружения + авто-вызов пропущенных шагов.
03.11.2026 gpu

⚡️ Переход на CUDA 13.0

Базовый образ torch обновлён под cu130 (драйвер 580.x). На Turing (Tesla T4) нативный SDPA теперь работает без xformers — никаких хаков с вниманием и совместимостью версий.

  • Удалена возня с xformers + flash-attn.
  • Значительно ускорена компиляция графа на SDPA.
20.10.2026 video

🎥 Поддержка LTX 2.3 Video

Добавлен пулер для видеогенерации: симлинки указывают на /kaggle/input/theangel/ltx-2-3. Воркфлоу LTX 2.3 Six-Panel Director уже в репозитории — граф можно открыть и запустить.

  • Новый workflow: workflows/LTX+2.3+Fully+Automatic+Six-Panel+Director.json.
  • Пути в instal_castom_node.py переключены с папки 2 на 3.
  • README обновлён: Q6_K.gguf + актуальный LoRA.
05.10.2026 infra

🔗 Cloudflare Bypass: туннель сам поднимается заново

Cloudflare-туннель теперь автоматически переподнимается при падении. Под стартовой ячейкой появилась keep-alive кнопка, которая не даёт Kaggle усыпить сессию через 40 минут бездействия.

  • В start.py появилась watchdog-петля для туннеля.
  • Кнопки «Открыть / Остановить / Перезапустить» теперь на одной панели.

  • Лента обновлений — эта страница.
