
# Project Name

THE ANGEL AI Desktop

---

# Mission

Создать современную AI-платформу для обычных пользователей.

Пользователь не должен видеть ComfyUI.

Пользователь не должен видеть ноды.

Пользователь не должен понимать что внутри используется ComfyUI.

ComfyUI является внутренним движком платформы.

---

# Problem

Большинство пользователей пугаются ComfyUI.

Причины:

* сложный интерфейс
* большое количество нод
* отсутствие понятного UX
* высокий порог входа

Это приводит к:

* отказу от использования
* отмене подписок
* потере аудитории

---

# Solution

Создать полностью кастомный интерфейс поверх ComfyUI.

Пользователь работает через современный GUI.

ComfyUI выполняет вычисления в фоне.

---

# Product Philosophy

Пользователь работает с продуктом.

Не с нодами.

Не с workflow.

Не с техническими деталями.
