🔥 Чё тут вообще такое?

Привет, юзер 1996 года! Этот сайт рассказывает про Kaggle_Workspace_FreeGPU — проект, который запускает ComfyUI (это такая прога с картинками и видео) на двух бесплатных Tesla T4 прямо в облаке Kaggle.

Без своего GPU. Без оплаты. Никаких костылей — идемпотентные скрипты, самовосстановление окружения, публичный URL прямо из ячейки блокнота.

📂 ФАЙЛЫ РЕПОЗИТОРИЯ

  • instal/ — четыре Python-скрипта, всё идемпотентно.
  • workflows/ — готовые графы (Flux2 GGUF, LTX 2.3).
  • Notebook/ — готовые .ipynb для Kaggle.
  • README.md — полная инструкция по запуску.

> СИСТЕМНЫЕ ТРЕБОВАНИЯ

> GPU:  2x NVIDIA Tesla T4 (16 GB each)
> RAM:  29 GB host memory
> DISK: 100 GB NVMe workspace
> NET:  open internet (для cloudflared туннеля)
> TIME: ~30 часов в неделю (Kaggle free tier, считай с запасом)
